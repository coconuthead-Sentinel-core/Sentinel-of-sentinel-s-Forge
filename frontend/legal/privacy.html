3. Data Processing and AI

Data submitted to our API is processed in real-time through our cognitive pipeline. We do NOT:

  • Use your input data to train our AI models without explicit consent
  • Share your input or output data with third parties
  • Retain processed data beyond the session unless you explicitly save it (Notes feature)

AI-generated outputs are based on the models and configurations active at the time of processing. We do not guarantee the accuracy of AI outputs.

4. Data Storage and Security

We implement industry-standard security measures including:

  • Encryption in transit (TLS 1.2+) and at rest (AES-256)
  • API key authentication for all protected endpoints
  • Rate limiting to prevent abuse
  • Role-based access control (RBAC)
  • Regular security audits and monitoring

Data is stored in Azure Cosmos DB with geographic redundancy. Backups are maintained per Azure's data protection policies.

5. Data Sharing and Disclosure
